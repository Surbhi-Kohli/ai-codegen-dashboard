Navigation Structure

Sidebar Nav
  • Overview — KPI cards + summary charts
  • Delivery — velocity, throughput, cycle time
  • Bottlenecks — time-in-stage, queue, stuck items
  • AI Impact — contribution, review quality, tools
  • AI Quality & Oversight — defect rate, copy-paste flags, review depth
Drill-Down Pattern
  • Click any chart segment → filter to that dimension
  • Team view → click developer → developer detail
  • Developer detail → click issue → issue timeline
  • Issue timeline shows full stage-by-stage breakdown

Stretch: Webex Actions

  • Automated alerts — configurable via config file; post to review space when thresholds exceeded
  • "Nudge via Webex" button — on stuck items table; manager clicks to send manual reminder
  • Escalation to discussion channel — if no response in review space after threshold
